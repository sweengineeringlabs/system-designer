# System Designer Project

This project consists of two main components:

1.  **System Designer Architect:** A React/Rust-based tool for designing AI systems using an 8-step framework.
    - Located in `system-designer-agent/`.
    - Tech: React, Tailwind, Rust Axum, Tauri.

2.  **Browser Agent (Antigravity):** A high-performance autonomous browser control agent.
    - Located in `browser-agent/`.
    - Tech: Rust, Chromiumoxide, Gemini 2.0 Flash.

## Documentation Structure

- `/doc`: Project-wide documentation.
- `/browser-agent/doc`: Specific documentation for the browser agent.
- `/system-designer-agent/doc`: Specific documentation for the architect tool.
