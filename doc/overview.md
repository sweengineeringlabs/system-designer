# Project Overview

This document provides a detailed overview of the System Designer project.

## Components

### 1. System Designer Architect
- **Path:** `system-designer-agent/`
- **Tech Stack:** React, Tailwind, Rust (Axum), Tauri.
- **Description:** A tool for designing AI systems using a structured 8-step framework.

### 2. Browser Agent (Antigravity)
- **Path:** `browser-agent/`
- **Tech Stack:** Rust, Chromiumoxide, Gemini 2.0 Flash.
- **Description:** A high-performance autonomous browser control agent designed for complex web tasks.

## Getting Started

Refer to the specific documentation in each component's directory for installation and usage instructions.
